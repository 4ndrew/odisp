Классы обеспечивающие стандартные сообщения ядра ODISP:
  • Работа с объектами
    1. od_object_loaded -- сообщение о запуске объекта
    2. od_cleanup -- останов объекта
    3. od_load_object -- подгрузка внешнего объекта
    4. od_unload_object -- принудительная выгрузка объекта
    5. od_shutdown -- завершение работы диспетчера
    6. od_list_objects -- список подгруженых объектов
  • Работа с ресурсами
    1. od_list_resources -- выдать список ресурсов
    2. od_acquire -- запрос на захват ресурса
    3. od_release -- запрос на высвобождение ресурса
